# egg-dva-umi

It's a scaffold. Use [eggjs][egg], [umijs][umi] and [dvajs][dva] to create a frontend and backend project

### QuickStart

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### Build

```bush
$ npm run build
```

[egg]: https://eggjs.org
[umi]: https://umijs.org
[dva]: https://dvajs.com
